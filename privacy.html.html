隱私政策

最後更新日期：2024年1月

1. 信息收集

本平台尊重您的隱私，我們收集的信息包括：

  • 您提供的API密鑰（僅在本地瀏覽器存儲）
  • 使用統計數據（請求次數、API使用情況）
  • 技術信息（瀏覽器類型、IP地址）

2. API密鑰安全

我們採用以下措施保護您的API密鑰：

  • 所有API密鑰僅在您的本地瀏覽器中加密存儲
  • 平台服務器從不存儲或訪問您的API密鑰
  • 使用HTTPS加密傳輸所有數據

3. 數據使用

我們收集的數據僅用於：

  • 提供AI蒸餾服務
  • 改善平台性能
  • 生成使用統計報告

4. 第三方服務

本平台整合以下第三方AI服務：

  • OpenAI API - 隱私政策
  • Google Gemini - 隱私政策
  • DeepSeek API - 隱私政策
  • Anthropic Claude - 隱私政策

5. Cookie使用

我們使用必要的Cookie來：

  • 保存用戶設置
  • 記住API密鑰設置
  • 提供更好的用戶體驗

6. 您的權利

您有權：

  • 訪問您的數據
  • 刪除本地存儲的數據
  • 停止使用服務

7. 政策變更

我們可能不定期更新本隱私政策，更新後會在本頁面發布通知。

8. 聯繫我們

如有任何隱私相關問題，請聯絡：contact@aifusionpro.com

重要提醒：您的API密鑰是您與AI服務商之間的直接關係，請妥善保管並定期更換。
